# Pharmacy Management System

A web-based/application system designed to manage pharmacy operations such as inventory, sales, and customer records.

## Features
- Add / update / delete medicines
- Manage stock inventory
- Handle customer records
- Sales tracking system
- Admin dashboard

## Technologies Used
- Python
- Database (MySQL / SQL Server)

## Purpose
This project was built to practice database management systems and real-world CRUD application development.

## Future Improvements
- Add authentication system
- Improve UI design
- Add reporting/analytics dashboard
